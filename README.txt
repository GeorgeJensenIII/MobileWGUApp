MobileWGUApp:

By: George W. Jensen III

Last Update: 04/07/19

About:

Mobile WGU App is an android application that was to allow students to track their terms, courses, assessments, mentors, and course notes.

The app features interactive screens where users can view each of their terms, courses, assessments, mentors and add, modify and delete from these lists.

The app is totally stand alone, however could be adapted to work with school databases in order to pull live data.

Navigation:

All screens utilize the android back button, as well as other interactive buttons on the menu bar.

When viewing items in a list, long pressing an item will change the selection mode and a trash can will appear in the top right side of the menu bar to allow users to delete selected items from the list.
